# Maskesiz Çalışan Tespiti

Python + OpenCV + YOLO kullanarak yapılan, çalışanların maske takıp takmadığını analiz eden bir proje.
Her çalışan için maskeli/maskesiz tespiti yapar ve sonuçları görselleştirir.

## Demo
![Demo](demo)  # görseller

## Özellikler (Features)
- Çalışan başına maskeli/maskesiz tespit
- YOLOv5 modeli ile gerçek zamanlı tespit
- Özel olarak toplanmış veri seti ile eğitim ve analiz
- OpenCV ile görüntü işleme
- PyCharm ile analiz ve sonuç görselleştirme
